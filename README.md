# Premium ToDo App

A modern, visually stunning ToDo application built with Vanilla HTML, CSS, and JavaScript. Featuring a premium "Glassmorphism" design with smooth animations and local persistence.

## Features

- **Modern UI**: Glassmorphism effect with animated background blobs.
- **Micro-animations**: Smooth transitions for adding, completing, and deleting tasks.
- **Persistence**: Tasks are saved to your browser's LocalStorage.
- **Filtering**: View All, Active, or Completed tasks.
- **Responsive**: Works on desktop and mobile browsers.

## Technologies Used

- HTML5
- CSS3 (Vanilla CSS with Custom Variables)
- Vanilla JavaScript (ES6+)
- Google Fonts (Outfit)

## How to Run

Simply open `index.html` in any modern web browser.
